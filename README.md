# 🛍️ Lifestyle Store – E-commerce Website

**Lifestyle Store** is a modern and responsive front-end e-commerce website designed for selling lifestyle products such as fashion, electronics, accessories, home décor, and more. Built using pure HTML, CSS, and JavaScript, it offers a clean and user-friendly interface perfect for showcasing retail collections.

---

## 🌐 Live Demo

>https://github.com/amanprakash52/E-Commerce-Website

---

## 🛠️ Features

- 👗 Home page with featured lifestyle products
- 🛒 Product catalog/grid layout
- 📱 Fully responsive design for all devices
- 🧭 Navigation bar with categories
- 🔍 Search bar (UI only)
- 🖼️ Image slider/banner
- 💳 Cart and Buy Now buttons (UI)
- ✉️ Contact form
- 🎨 Clean, modern design using HTML5, CSS3, and JavaScript
